# PDF Genius

A powerful, client-side PDF toolkit — part of the [Genius Space](https://genius-space.org) ecosystem.

**🌐 Live:** [pdf.genius-space.org](https://pdf.genius-space.org)

## Workflow

Upload one or more PDFs → edit everything in a unified page grid → download the final result.

All tools work together in a single view — merge multiple files, rearrange, crop, annotate, watermark, then download one combined PDF.

## Features

### 📑 Merge & Reorder
- Drag & drop multiple PDFs to combine them
- Drag page thumbnails to reorder across files
- Add more PDFs at any time

### 📄 Page Management
- Visual thumbnail grid of all pages
- Select & remove unwanted pages
- Rotate pages (90° increments)
- Source file labels on each page

### ✂️ Crop
- Visual crop editor with draggable handles
- Precise crop dimensions in PDF points
- Per-page cropping

### ✏️ Annotate & Draw
- Freehand drawing tool
- Highlighter tool
- Text annotations
- Eraser for removing annotations
- All annotations are embedded into the final PDF

### 📋 Fill PDF Forms
- Auto-detects AcroForm fields (text, checkbox, dropdown, radio, option list)
- Visual overlay editor — fields positioned exactly over the rendered PDF
- Multi-page form navigation
- Values are baked into the PDF on download (form flattened)

### 🔏 Watermark
- Add text watermark to every page
- Configurable font size, rotation, color & opacity

## Tech Stack

- **pdf-lib** — PDF creation & modification
- **PDF.js** — PDF rendering & page thumbnails
- **Font Awesome** — Icons
- **Vanilla JS** — No frameworks, no build step

## Privacy

100% client-side. No files are uploaded. All processing happens in your browser.
